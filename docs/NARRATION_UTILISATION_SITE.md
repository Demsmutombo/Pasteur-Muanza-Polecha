# Narration Explicative - Utilisation du Site Memorial Days

## Introduction
Bienvenue sur le site mémorial dédié à **Pasteur Muanza Polecha Matunga Zebedee**. Ce site permet de découvrir sa vie, consulter des témoignages, parcourir des galeries photo et télécharger une carte mémoriale. Cette narration vous guide étape par étape dans l'utilisation de toutes les fonctionnalités.

## ÉTAPE 1 : Arrivée sur le Site et Première Impression

### 1.1 Page de Chargement
Lorsque vous ouvrez le site, vous verrez :
- **Un écran de chargement** qui apparaît brièvement pendant le chargement des ressources
- Le site se prépare à afficher le contenu principal

### 1.2 Sélecteur de Thème
En haut à droite de la page, vous trouverez un **sélecteur de thème** qui permet de basculer entre :
- Le thème clair
- Le thème sombre
- Le thème personnalisé

**Comment l'utiliser :** Cliquez sur l'icône de thème pour changer instantanément l'apparence du site selon vos préférences.

### 1.3 Menu de Navigation
En haut de la page, le **menu de navigation** est toujours visible et comprend :
- **Accueil** : Retour à la page principale
- **Services** : Section des vidéos des obsèques
- **Galerry** : Galerie de photos
- **Carte Mémorial** : Page pour télécharger la carte commémorative
- **Contact** : Informations de contact

**Sur mobile :** Un bouton hamburger (☰) permet d'ouvrir/fermer le menu.

## ÉTAPE 4 : Utilisation de la Page Carte Mémorial

### 4.1 Accès à la Page
Pour accéder à la carte mémorial :
- Cliquez sur **"Carte Mémorial"** dans le menu de navigation
- Tapez `/carte-memorial` dans l'URL

### 4.2 Visualisation de la Carte
La carte mémorial s'affiche avec :
- **Fond noir élégant** avec bordures dorées
- **Décorations florales** aux quatre coins et sur les côtés
- **Image circulaire** du Pasteur avec bordure dorée
- **Nom complet** : Pasteur Muanza Polecha Matunga Zebedee
- **Dates** : 15 Mai 1992 - 2 Septembre 2024
- **Message commémoratif**
- **Informations du service** : Date et lieu de sépulture

### 4.3 Téléchargement de la Carte
**Processus étape par étape :**

1. **Faites défiler** jusqu'au bouton **"Télécharger la carte"** en bas de la page
2. **Cliquez** sur le bouton
3. **Attendez** que le message "Génération en cours..." apparaisse
4. La carte est **convertie en image haute qualité** (résolution 3x pour une qualité optimale)
5. Le **téléchargement démarre automatiquement** avec le nom : `carte-memorial-pasteur-muanza-polecha.png`

**Temps d'attente :** Quelques secondes selon votre connexion et la puissance de votre appareil.

**Format :** L'image est sauvegardée en PNG haute résolution, parfaite pour :
- Impression
- Partage sur réseaux sociaux
- Envoi par email ou messagerie

**Note importante :** Si le téléchargement échoue, réessayez. Le bouton retrouvera son état normal après l'opération.

## ÉTAPE 7 : Résolution de Problèmes Courants

### 7.1 La Page Ne Charge Pas
**Solutions :**
- Rafraîchissez la page (F5 ou Ctrl+R)
- Vérifiez votre connexion internet
- Vide le cache de votre navigateur

### 7.2 Les Vidéos Ne S'Affichent Pas
**Solutions :**
- Vérifiez que JavaScript est activé
- Les vidéos s'ouvrent sur YouTube : assurez-vous d'avoir accès à YouTube
- Vérifiez que les pop-ups ne sont pas bloquées

### 7.3 Le Téléchargement de la Carte Échoue
**Solutions :**
- Réessayez après quelques secondes
- Vérifiez l'espace disponible sur votre appareil
- Utilisez un autre navigateur (Chrome, Firefox, Edge recommandés)

### 7.4 Le Menu Mobile Ne S'Ouvre Pas
**Solutions :**
- Appuyez sur le bouton hamburger (☰)
- Rafraîchissez la page
- Vérifiez que JavaScript est activé
