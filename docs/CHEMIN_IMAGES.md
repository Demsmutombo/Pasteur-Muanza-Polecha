## Notes importantes:
1. Les chemins doivent utiliser des espaces dans les noms de dossiers (ex: "image du pasteur")
2. Les noms de fichiers contiennent des espaces et des caractères spéciaux
3. Les extensions sont `.jpeg` et non `.jpg`
4. Pour les URLs dans le code HTML/Vue, les espaces seront automatiquement encodés par le navigateur
